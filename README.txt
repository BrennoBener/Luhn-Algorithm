In this project, i will implement the Luhn Algorithm. 
This algorithm is a formula to validate a variety of identification numbers.
The Luhn algorithm is as follows:
From the right to left, double the value of every second digit; if the product is greater than 9,
 sum the digits of the products.
Take the sum of all the digits.
If the sum of all the digits is a multiple of 10, then the number is valid; else it is not valid.

Nesse projeto, eu vou implementar o algoritmo de Luhn.
Este algoritmo é uma formula para validar uma variedade de numeros de verificação.
O algoritmo de Luhn é o seguinte:
Da direita para a esquerda, dobre o valor de cada segundo dígito; se o produto for maior que 9, 
some os dígitos dos produtos.
Pegue a soma de todos os dígitos.
Se a soma de todos os dígitos for múltiplo de 10, então o número é válido; caso contrário,
